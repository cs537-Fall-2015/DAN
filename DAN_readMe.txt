+Extra JARS Needed:
-gson 2.4

+Created libs folder and added JARS
+Added JARS to Eclipse

--select add external JARS... -> select JARS from libs folder and add both JARS

+To run the Program:
-navigate to src -> DAN.DAN_testMain
-run file DAN_testMain.java
